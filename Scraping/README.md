# Question 1 - Scraping

## Dependencies

1. pandas
2. BeautifulSoup
3. requests

### Running the code

```
python3 scraper.py [url]
python3 jsonmatch.py [csv file] [json file]
```

### Examples

```
python3 scraper.py https://summerofcode.withgoogle.com/archive/2019/projects/
python3 gsoc2019.csv students.json
```
